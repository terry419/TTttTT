최종 설계도: CurseExplosionManager.cs
파일 이름: CurseExplosionManager.cs

타입: MonoBehaviour (씬에 존재하는 오브젝트에 부착)

설명: 모든 몬스터의 죽음을 감지합니다. 만약 죽은 몬스터가 특정 '저주' ID를 가진 상태이상을 가지고 있었다면, 그 자리에 설정된 폭발을 일으킵니다.

인스펙터 필드 명세
필드명	타입	UI 형태	설명
[ 감지할 저주 설정 ]			
Curse Status Effect ID	string	문자열 입력	폭발을 유발하는 '저주' 효과의 Status Effect ID를 지정합니다. 카드 모듈에서 설정한 ID와 정확히 일치해야 합니다. (예: "죽음의 표식")
[ 폭발 효과 설정 ]			
Explosion Radius	float	숫자 입력	폭발이 주변 적에게 피해를 주는 범위 (반지름).
Explosion Damage Type	Enum	드롭다운	Flat (고정 수치), MaxHealthPercentage (최대 체력 비례 %)
Explosion Damage Amount	float	숫자 입력	폭발의 기본 피해량.
Scales With Caster Damage Bonus	bool	체크박스	체크 시, 저주를 건 플레이어의 FinalDamageBonus 스탯이 폭발 피해량에 영향을 줍니다.
Explosion VFX	AssetReferenceGameObject	오브젝트 필드	폭발이 일어날 때 생성될 시각 효과(VFX)의 어드레서블 주소.

Sheets로 내보내기
전체 동작 방식
카드 제작: ApplyEffectToHitTargetModule을 사용하여 Status Effect ID가 "죽음의 표식"인 카드를 만듭니다.

관리자 설정: 씬에 있는 CurseExplosionManager의 Curse Status Effect ID 필드에 똑같이 "죽음의 표식"이라고 입력하고, 폭발 범위와 피해량을 설정합니다.

게임 플레이: 플레이어가 카드로 적을 공격하면, 적에게 "죽음의 표식" 효과가 부여됩니다.

효과 발동: 이후 그 몬스터가 어떤 이유로든 죽으면, CurseExplosionManager가 이를 감지하고 설정된 폭발을 일으켜 주변 적에게 피해를 줍니다.
